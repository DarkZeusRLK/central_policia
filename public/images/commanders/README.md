# Fotos dos Comandantes

Esta pasta deve conter as fotos dos comandantes em formato JPG.

## Estrutura de Nomes:

### Comando Geral (index.html):
- `comando_geral_1.jpg` - Comandante Geral
- `comando_geral_2.jpg` - Subcomandante Geral (1º)
- `comando_geral_3.jpg` - Subcomandante Geral (2º)

### PCERJ (departamentopcerj.html):
- `pcerj_1.jpg` - Delegado Geral
- `pcerj_2.jpg` - Delegado Adjunto (Operacional)
- `pcerj_3.jpg` - Delegado Adjunto (Administrativo)

### PMERJ (batalhaopmerj.html):
- `pmerj_1.jpg` - Comandante
- `pmerj_2.jpg` - Subcomandante (Gestão)
- `pmerj_3.jpg` - Subcomandante (Planejamento)

### PF (departamentopf.html):
- `pf_1.jpg` - Diretor Geral
- `pf_2.jpg` - Vice-Diretor (Administrativo)
- `pf_3.jpg` - Vice-Diretor (Fiscalização)

### PRF (departamentoprf.html):
- `prf_1.jpg` - Diretor Geral
- `prf_2.jpg` - Vice-Diretor
- `prf_3.jpg` - Chefe de Policiamento

## Nota:
Se uma imagem não existir, o sistema automaticamente usará a foto do Discord como fallback.
