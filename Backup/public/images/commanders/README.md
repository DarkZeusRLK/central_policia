# Fotos dos Comandantes

Esta pasta deve conter as fotos dos comandantes em formato PNG.

## Estrutura de Nomes:

### Comando Geral (index.html):
- `comando_geral_1.png` - Comandante Geral
- `comando_geral_2.png` - Subcomandante Geral (1º)
- `comando_geral_3.png` - Subcomandante Geral (2º)

### PCERJ (departamentopcerj.html):
- `pcerj_1.png` - Delegado Geral
- `pcerj_2.png` - Delegado Adjunto (Operacional)
- `pcerj_3.png` - Delegado Adjunto (Administrativo)

### PMERJ (batalhaopmerj.html):
- `pmerj_1.png` - Comandante
- `pmerj_2.png` - Subcomandante (Gestão)
- `pmerj_3.png` - Subcomandante (Planejamento)

### PF (departamentopf.html):
- `pf_1.png` - Diretor Geral
- `pf_2.png` - Vice-Diretor (Administrativo)
- `pf_3.png` - Vice-Diretor (Fiscalização)

### PRF (departamentoprf.html):
- `prf_1.png` - Diretor Geral
- `prf_2.png` - Vice-Diretor
- `prf_3.png` - Chefe de Policiamento

## Nota:
Todas as imagens devem estar em formato PNG. O sistema usa apenas as imagens locais, sem fallback para Discord.
